Proyectos

Plataforma de E-commerce

Este proyecto es una plataforma de comercio electrónico diseñada para ofrecer una experiencia de compra fluida y agradable. La interfaz se centra en la experiencia del usuario, con un diseño limpio y minimalista. La plataforma permite a los usuarios navegar por productos, añadir artículos al carrito de compras y realizar transacciones seguras. El diseño es completamente responsive, asegurando una experiencia óptima tanto en dispositivos móviles como en ordenadores. Tecnologías utilizadas: JavaScript, React, Node.js, MongoDB.

Tecnologías utilizadas: JavaScript, React, Node.js, MongoDB.

Blog Personal

Este proyecto es una plataforma de blog personal con un diseño dinámico y atractivo. La interfaz incluye una sección para entradas destacadas, una barra lateral con las publicaciones más recientes y etiquetas, y una imagen de cabecera llamativa. El blog ofrece una experiencia de usuario suave, con animaciones al desplazarse y opciones para compartir en redes sociales. Cada entrada tiene una vista detallada, optimizada tanto para SEO como para usabilidad.

Tecnologías utilizadas: Python, Flask, HTML, CSS, Bootstrap.
